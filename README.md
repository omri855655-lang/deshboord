# דשבורד אקסל ללא שמירת מידע

הפרויקט הזה מעלה קובץ אקסל בכל שימוש, מעבד אותו בזיכרון בלבד, ומחזיר לדפדפן JSON שממנו נבנה הדשבורד. אין שמירה למסד נתונים ואין כתיבה של קובץ האקסל לשרת.

## מה הדשבורד יודע לעשות

- הצגה לפי מוסדות אקדמיים
- הצגה לפי מחוזות ואזורים גיאוגרפיים
- סינון לפי תחומי הכשרה
- תצוגת מפה סכמטית לפי ערים/מחוזות
- סיכום אגרגטיבי וגרפים לפי הסינונים שנבחרו
- חיתוך נפרד לפי קבוצות יישובים ממשלתיות שמוגדרות ב-[config/government_groups.json](/Users/omrigabay/Documents/New%20project/config/government_groups.json)

## פרסום על דומיין

הפרויקט מוכן עכשיו גם לפריסה כשירות ווב.

### Render

1. מעלים את הפרויקט ל-GitHub.
2. ב-Render יוצרים `Blueprint` חדש מתוך הריפו.
3. Render יקרא את [render.yaml](/Users/omrigabay/Documents/New project/render.yaml) וירים את השירות.
4. אחרי שהשירות עולה, נכנסים ל-`Settings > Custom Domains`.
5. מוסיפים את הדומיין שלך, למשל `dashboard.yourdomain.co.il`.
6. מעדכנים אצל ספק הדומיין את רשומת ה-DNS ש-Render מציג.

### מה חשוב לדעת

- גם בגרסת דומיין המערכת לא שומרת את האקסל במסד נתונים.
- הקובץ נשלח לשרת, מעובד בזיכרון, ומוחזר לדפדפן.
- אם תרצי הקשחה נוספת, אפשר בשלב הבא להוסיף מחיקה יזומה של קבצים זמניים, הגבלת גודל קובץ, וסיסמה לכניסה.

## התאמה למסמכי הממשלה

קובץ [config/government_groups.json](/Users/omrigabay/Documents/New%20project/config/government_groups.json) מחזיק את קבוצות היישובים הממשלתיות. כרגע הוא כולל קבוצות ברירת מחדל שנגזרו מהמסמכים שסיפקת, וניתן להרחיב אותו בקלות אם תרצה לדייק את כל רשימות היישובים מתוך נספחים מלאים.
